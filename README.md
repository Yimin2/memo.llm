# 🤖 AI 메모 웹 애플리케이션

Gemini API를 활용한 지능형 메모 관리 시스템

## 📋 프로젝트 소개

사용자의 자연어 입력을 AI가 분석하여 구조화된 메모로 자동 생성해주는 React 기반 웹 애플리케이션입니다.
"내일 오후 3시에 팀 미팅"과 같은 자연스러운 문장을 입력하면, AI가 제목, 내용, 카테고리, 마감일 등을 자동으로 분류하여 체계적인 메모를 만들어줍니다.

## ✨ 주요 기능

### 🎯 지능형 메모 생성
- 자연어 입력을 AI가 분석하여 구조화된 메모로 변환
- 제목, 내용, 카테고리, 우선순위, 마감일 자동 추출
- Gemini API를 활용한 정확한 의도 파악

### 🔐 사용자 인증
- Supabase 기반 회원가입/로그인 시스템
- 안전한 사용자 데이터 관리
- Redux를 통한 전역 인증 상태 관리

### 📝 메모 관리
- 전체/미완료/완료 메모 필터링
- 메모 상태 변경 (완료/미완료)
- 메모 삭제 기능
- 로컬 스토리지 기반 데이터 영속성

## 🛠️ 기술 스택

### Frontend
- **React** - UI 라이브러리
- **React Router v7** - 라우팅 관리
- **Redux Toolkit** - 상태 관리
- **Redux Persist** - 상태 영속화
- **Tailwind CSS** - 스타일링

### Backend & Services
- **Google Gemini API** - AI 자연어 처리
- **Supabase** - 사용자 인증 및 데이터베이스

## 📁 프로젝트 구조

```
memo.llm/
├── src/
│   ├── components/     # 재사용 가능한 컴포넌트
│   ├── layouts/        # 레이아웃 페이지
│   ├── pages/          # 페이지 컴포넌트
│   ├── router/         # router 경로
│   ├── store/          # Redux 스토어 설정
│   └── utils/          # 유틸리티 함수
├── public/
└── .env                # 환경 변수 (gitignore)
```


## 📄 페이지 구성

| 페이지 | 경로 | 설명 |
|--------|------|------|
| 서비스 소개 | `/` | 프로젝트 목적 및 기능 소개 |
| 로그인 | `/login` | 사용자 로그인 |
| 회원가입 | `/signup` | 신규 사용자 등록 |
| 메모 생성 | `/memo/create` | AI 기반 메모 생성 |
| 메모 목록 | `/memo/list` | 메모 조회 및 관리 |

## 🔒 보안
